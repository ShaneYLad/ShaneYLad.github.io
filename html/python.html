Python Projects

A collection of practical, security-focused tools built with Python — from Discord automation to digital hygiene and network scanning.

What I’ve Built

  • DedSec Discord Bot: Offers tech support, password strength checks, and breach detection in real-time.
  • Password Analyzer: Evaluates password strength and provides actionable feedback.
  • Port Scanner: Lightweight CLI tool to scan open ports on local or remote systems.
  • Cybersecurity Quiz: A terminal-based quiz to test and teach core security concepts.
  • Digital Declutter: Cross-platform cleanup script (PowerShell + Bash) that:
    • Clears browser cache (Chrome, Brave, Edge, Firefox, Safari)
    • Deletes browser history
    • Empties system temp folders
    • Empties Recycle Bin / Trash
    • Clears recent file history
    • Uses color-coded output and built-in safety checks

Libraries & Tools

Python, discord.py, requests, socket, os, shutil, and PowerShell/Bash scripting. Focused on portability, clarity, and user safety.

What I’ve Learned

Sharpened my skills in automation, CLI design, API integration, and cross-platform scripting. These projects reflect my passion for privacy, security, and clean system workflows.
