Virtual Machines

Used virtual machines as safe, isolated environments to explore Linux, test tools, and practice cybersecurity techniques.

What I’ve Done

  • Practiced installing and configuring Linux distros in VMs before setting them up on real hardware.
  • Explored Kali Linux and its tools like Wireshark, John the Ripper, Sherlock, and more.
  • Used VMs to safely test scripts, simulate attacks, and experiment with cybersecurity tools.
  • Created isolated environments for learning without risking my main system.

Tools Used

VirtualBox and VMware Workstation for VM management. Kali Linux for security testing, with tools like Wireshark, Hydra, John, and Sherlock.

What I Learned

Built confidence in Linux setup and troubleshooting, learned how to safely explore offensive tools, and developed a workflow for testing without breaking anything important.
