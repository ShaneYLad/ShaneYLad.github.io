DedSec Discord Bot

A Python-based Discord bot inspired by the DedSec aesthetic — built to empower users with tech knowledge, privacy tools, and security checks.

What I Built

  • A tech support bot with categorized advice on hardware, software, Linux, and privacy.
  • Password strength checker to help users create secure, resilient credentials.
  • Email breach scanner using external APIs to detect compromised accounts.

Tech Stack

Python, discord.py, requests, and third-party APIs for breach and password analysis. Designed with modular commands and a sleek DedSec-inspired command style.

What I Learned

Deepened my Python skills, especially with asynchronous programming and API handling. Learned how to design user-friendly command structures and deliver real-time security tools in a Discord environment.
